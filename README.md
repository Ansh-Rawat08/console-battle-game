#  Text-Based RPG Game (C++ OOP Project)

##  Overview
This is a console-based RPG game built using Object-Oriented Programming in C++.  
The player fights different enemy types with attack, defense, and special abilities.

##  Features
- Player vs Enemy combat system
- Multiple enemy types (Newbie, Medium, Boss)
- Special attacks and defense mechanics
- Creative Mode (free play)
- Manual Mode (level progression)
- Smart pointer memory management (no memory leaks)

##  Concepts Used
- Inheritance
- Polymorphism (virtual functions, overriding)
- Encapsulation

##  Game Modes
### 1. Creative Mode
- Player selects enemy level manually
- Endless gameplay until defeat

### 2. Manual Mode
- Fixed progression (Level 1 → 3)
- Player heals after each win
